# Commands

Leader `<space>`

## Search

To search a word that you have your cursor on:

```
* (shift + 8)
```

To cancel the hightlight if a search:

```
leader + nh
```

Navigation in wezterm:

Create new window horizontal:

```
cmd + d
```

Create new window vertical:

```
shift + cmd + d
```

Shift between windows

```
shift + cmd + <arrow key>
```
